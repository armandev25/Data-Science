# Week 2 - Student Grade Calculator

## Project Overview
This project is a Python-based Student Grade Calculator that determines a student's grade based on their marks and provides encouraging feedback.

---

## Objectives

- Understand if-elif-else decision making
- Implement while loops
- Use functions for modular programming
- Handle invalid inputs using validation
- Practice error handling with try-except

---

## Setup Instructions

1. Install Python 3.x
2. Navigate to project folder
3. Run:

   python grade_calculator.py

---

## Project Structure

- grade_calculator.py → Main program
- test_cases.txt → Testing examples
- screenshots/ → Program output screenshots

---

## Grading Logic

A: 90–100  
B: 80–89  
C: 70–79  
D: 60–69  
F: 0–59

✔ if-elif-else grading logic

# Week 1: Personal Introduction Program

## Project Overview
This project is a beginner-level Python program that collects user information and displays a personalized welcome message.

## Objectives
- Learn variables
- Use input() function
- Use print() statements
- Practice f-strings
- Understand program execution flow

## Setup Instructions

1. Install Python 3.x
2. Run:
   python personal_intro.py
